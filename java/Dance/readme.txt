※PCにjavaの環境がない場合に上記の操作を実行しても「JavaVMを〜」って出てきて実行できないようです。
ORACLEのサイトでJavaのダウンロードが必要なようです。かなすぃ。
https://www.java.com/ja/download/　(ORACLE URL)



隊列表を作る時に簡易にできるソフトです。
時間がないときに活用してもらえればと思います。

オリジナリティ等出したい場合は利用することをお勧めしません。


操作方法：
Ctrl + s ...保存
Ctrl + n ...新規名前登録

PClear...サークル消去
MClear...矢印消去
TClear...テキスト消去
